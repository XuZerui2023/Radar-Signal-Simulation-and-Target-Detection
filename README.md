# Radar-Signal-Simulation-and-Target-Detection
该项目用于仿真多脉冲多波束线性相控阵雷达的（多）目标回波生成以及信号处理算法，最后进行目标检测和目标点参数测量。简要流程：信号回波模拟->信号处理->数据处理->可视化分析
This project simulates (multi-)target echo generation and signal processing algorithms for a multi-pulse multi-beam linear phased array radar, culminating in target detection and parameter measurement.
Workflow:Signal Echo Simulation $\rightarrow$ Signal Processing $\rightarrow$ Data Processing $\rightarrow$ Visualization & Analysis
