# 程序说明：
* 本程序仿真多脉冲多波束线性相控阵雷达的（多）目标回波生成以及信号处理算法，最后进行目标点参数测量。
* 仿真程序与实测数据处理程序不同的点在于，仿真中未考虑各脉冲的盲距，各采样波门采样点目标距离均是从0开始（实测数据中应该是从各自盲距后开始）

# 项目文件说明

## 1. 主程序算法（程序）步骤
文件名：`main_simulate_echoes_with_array_vx.m`，注意 `vx` 是程序版本号
1. 基本雷达系统参数定义
2. 混合三脉冲（窄中长）发射信号生成
3. 根据发射信号和目标数据仿真生成16通道（线性相控阵阵元通道）的目标回波信号
4. 根据设计好的天线方向图进行数字波束成形（DBF），将16通道信号数据加权生成具有方向性的13波束数据
5. 对多脉冲多波束的信号数据矩阵进行脉冲压缩处理，提取距离维信息
6. 对多脉冲多波束的信号数据矩阵进行MTD处理，多prt进行积累，提取速度维信息
7. 对MTD处理后的多波束 Range-Velocity (RD) 数据三维矩阵进行和波束 2D CA-CFAR（独立十字形，选大）
8. 对和波束CFAR目标检测后的初步检测点进行参数测量（距离插值、速度插值、和差波束单脉冲比幅测角）
9. 对进行参数测量后的初步检测点进行二级聚类/波束归并，第一级进行波束内聚类（策略：功率加权平均），第二级进行波束间聚类（策略：赢者通吃选大）
10. 最终可视化分析

## 2. 天线方向图仿真算法（程序）步骤
文件名：`plot_beam_patterns.m`
* 本程序用于加载给定DBF系数并绘制13个雷达波束方向图

## 3. 多波束对鉴角曲线（单脉冲测角K值）仿真算法（程序）步骤
文件名：`calibrate_monopulse_slopes.m` 和 `calibrate_all_monopulse_slopes.m`
* 本程序用于加载给定DBF系数和天线方向图仿真生成每个波束对的鉴角曲线（单脉冲测角K值）

## 4. MUSIC阵列信号高分辨测角
文件名：`MUSIC_1D.m` 和 `MUSIC_2D.m` 
* 一维MUSIC空间谱估计和二维MUSIC空间谱估计
